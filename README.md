# HealthVerse Portal
https://health-verse-portal.vercel.app/

HealthVerse Portal is a multi-purpose health and medical care system.
It opens the "portal" to a trio of tools, HealthSense AI, BloodLink Kerala and Hydraweight.

# HealthSense AI 
https://health-sense-ai-three.vercel.app/

HealthSense AI is an advanced tool that uses OCR and LLM analysis to to decode bills and prescriptions. It provides a price breakdown of the billed items which lets the user understand where they may have been overcharged. It uses RAG to keep an index of prices which is ultimately compared. Prescription analyis is supported which breaks down the purposes of medications along with any unwanted interactions or precautions to be followed while consuming the prescribed drugs.

# BloodLink Kerala
https://bloodlink-gamma.vercel.app/

BloodLink Kerala is a tool that connects people in need of blood with donors and bloodbanks. One can contact a bloodbank with a single tap for enquiry. It also educates the users about blood types and their specialities. We can click on a blood group to filter by availabilty and location.

# Hydraweight
https://hydraweight.vercel.app/

Hydraweight is a weight and water consumption tracking tool that aims to promote better life and wellness through proper hydration remainders and plotted out weight charts. It provides the option for creating and managing multiple user accounts.
